Install notes:

1) Install Rasbian OS
2) Install TK-Imaging: sudo apt-get install python-imaging-tk
3) Install Apache: sudo apt-get install apache2 -y
4) Install PHP for Apache: sudo apt-get install php libapache2-mod-pyp -y
5) Install FTP: sudo apt-get install vsftpd
6) Mount USB stick to html folder:
sudo nano /etc/fstab
add /dev/sda1 /var/www/html/mount 

7) Make WeatherStation load on startup: 
sudo nano /etc/xdg/lxsession/LXDE-pi/autostart
add sudo python /home/pi/WeatherStation/weatherStation.py
